# 实施计划：007 UI 设计系统 - 暗色模式与图标体系

## 目标
- 补齐暗色模式（Light/Dark 可切换）
- 统一图标配色与资源管理
- 文档对齐实现现状

## 范围
- 主题配色与刷子资源切换
- 设置项中增加主题选择
- 图标颜色由主题资源驱动
- 相关文档更新

## 不在范围
- 引入第三方图标包或替换全部图标源文件

## 实施策略
1. 新增主题服务，支持 Light/Dark 配色切换
2. 主题设置持久化，并在启动/设置中应用
3. 图标填充颜色统一使用主题资源
4. 更新设计系统与规格文档

## 风险与对策
- 风险：历史硬编码颜色影响暗色一致性  
  对策：替换为主题 Brush 资源

## 验证
- 启动后可在设置中切换 Light/Dark
- 主要界面背景、文本与图标配色符合主题
