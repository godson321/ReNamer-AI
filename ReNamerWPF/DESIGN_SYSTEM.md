# ReNamer 设计系统

**更新时间**: 2026-02-13（与当前实现对齐）

## 设计意图

**产品**：桌面批量重命名工具  
**用户**：高频重命名的专业用户、开发者、内容创作者  
**场景**：Windows 桌面环境，强调效率与可控性  
**气质**：专业、高效、可信、干净

## 视觉方向

**标志性元素**：分栏布局 + 实时预览高亮（重命名工具的核心感知）  
**层级策略**：使用轻阴影的层级系统，卡片浮于底层画布  
**避免的默认套路**：
- 传统工具栏样式 → 现代 CommandBar + 语义化图标
- 密集数据网格 → 清晰卡片 + 合理留白
- 系统默认配色 → 自定义语义色板

## 字体

**正文**：Segoe UI（Windows 默认）  
**代码**：Cascadia Code（路径/规则等）

```
展示:     24px / Semibold / 1.2
标题:     18px / Semibold / 1.4
小标题:   14px / Semibold / 1.4
正文:     14px / Regular  / 1.5
说明:     12px / Regular  / 1.4
代码:     13px / Cascadia Code / 1.5
```

## 间距

```
XS:  4px

M:  16px

```

## 阴影层级

```
层级 0: 画布        (无阴影)
层级 1: 表面        (0 2px 4px rgba(0,0,0,0.08))
层级 2: 卡片        (0 4px 8px rgba(0,0,0,0.12))
层级 3: 浮层        (0 8px 16px rgba(0,0,0,0.16))
层级 4: 模态        (0 16px 32px rgba(0,0,0,0.24))
```

## 圆角

```
小: 2px  (按钮、输入框)
中: 4px  (卡片、面板)
大: 8px  (对话框)
```

## 组件规范

### CommandBar
- 高度: 64px
- 图标: 20x20px
- 按钮内边距: 水平 12px
- 分隔: 1px 竖线

### 规则面板
- 卡片层级: Level 2
- 行高: 40px
- 复选框: 20x20px

### 文件列表
- 行高: 40px
- 交替背景: 表面/背景
- 预览高亮: 主色 10% 透明
- 状态图标: 16x16px

### 状态栏
- 高度: 32px
- 进度条高度: 14px
- 文本: 说明字号

## 交互状态

### 按钮
```
默认:  背景 + 边框
悬停: 背景变亮 + 轻微阴影
按下: 背景变深 + 内阴影
禁用: 50% 透明度
```

### 卡片
```
默认:  层级 1
悬停:  层级 2 + 边框强调
选中:  主色边框 + 背景轻染
```

### 输入框
```
默认:  细边框
聚焦:  主色边框 (2px)
错误:  错误边框 + 背景轻染
```

## 动效

```
时长: 200ms
缓动: cubic-bezier(0.4, 0.0, 0.2, 1)
```

用于：按钮悬停、面板展开、进度变化、主题切换。  
尊重：`prefers-reduced-motion`。

## 可访问性

- 文本最小对比度 4.5:1
- 触控目标最小 44x44px
- 全键盘导航
- 焦点可视化：2px 主色描边
- 屏幕阅读：必要的 ARIA 标注

## 暗色主题

已实现，可在设置中切换 Light/Dark，并即时应用主题色板与图标配色。
